# h5-mlb-archive

## 项目简介

h5-mlb-archive 是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，主要用于存放各类 H5 页面资源，便于长期保存和访问。

## 目录结构

```
h5-mlb-archive/
├── pages/          # 存放各个独立 HTML 页面文件
├── assets/         # 页面依赖的静态资源（图片、样式、脚本等）
├── docs/           # 项目相关文档或说明
└── README.md       # 本文件
```

## 页面归档说明

- 所有归档页面均以独立 HTML 文件形式存在，位于 `pages/` 目录下。
- 每个页面及其相关资源（如需）会保持相对路径引用，确保离线或移动后仍可正常显示。
- 归档内容仅作保存用途，不保证与原始来源完全一致。

## 维护说明

- 本仓库由维护者不定期更新，新增或修改页面时会提交对应变更。
- 欢迎通过 Issues 提交页面归档建议或报告链接失效问题。
- 不接受直接修改已有归档内容的 Pull Request，除非涉及明显错误或格式修正。

## 许可

本仓库内容仅供学习与存档使用，不构成任何形式的授权或担保。如需引用或分发，请自行评估合规性。
